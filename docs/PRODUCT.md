# Product: ReplyWise Local

**It does not just write replies. It tells you whether replying is a good idea.**

ReplyWise Local is a local-first co-pilot for WhatsApp and Telegram.

## Killer loop

```txt
Incoming message
↓
Should I reply?
↓
Smart Autopilot chooses best option
↓
User sends OR safe auto-send handles simple low-risk replies
↓
Browser agent sends via web UI
```

## MVP focus

- WhatsApp + Telegram only
- no official messaging API keys
- local rule engine by default
- no screenshot/OCR reading
- Smart Autopilot
- mobile-first decision cards
